# Notification Sounds

## notification.mp3

This file should contain a short notification sound (1-2 seconds).

### Recommendations:
- Format: MP3
- Duration: 1-2 seconds
- Volume: Normalized to prevent being too loud
- Sound: Pleasant, non-intrusive chime or bell tone

### Free sound resources:
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/
- https://notificationsounds.com/

### Example sounds to download:
1. "Notification Alert" - https://mixkit.co/free-sound-effects/notification/
2. "Positive Notification" - https://mixkit.co/free-sound-effects/alert/
3. Simple Bell Chime - Various free sound libraries

Place the downloaded/created notification.mp3 file in this directory.
